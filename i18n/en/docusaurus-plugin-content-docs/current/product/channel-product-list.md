# Channel Product List

From the left menu **Product → Channel Product List**, you can check **how products are displayed per channel**. While the Product List takes a product master perspective, this screen takes a **channel sales perspective**.

---

## Searching

| Search Item | Input Method |
|-----------|-----------|
| **SKU / SAP / Model / UPC Code** | Enter multiple entries separated by line breaks |
| **SAP Name** | Product name |
| **Channel** | Select multiple channels |
| **Channel Availability** | Whether the product is sellable on the channel — Yes / No |
| **Product Info Status** | Complete / Incomplete |
| **Product Type** | Single / Bundle |

### List Columns

| Column | Description |
|------|------|
| Product Type | Single / Bundle |
| **SKU Code** | Click to go to the [Product Detail](./product-detail) page |
| SAP Code / Quantity / SAP Name / Model Code / UPC Code | Basic product information |
| **Channel** | Channel name |
| **Channel Availability** | Whether the product is sellable on that channel (Yes/No) |
| **Product Info Status** | Whether product info is complete |

:::tip
If a product is not selling on a specific channel, check on this screen whether **Channel Availability** is No. Channel visibility is changed in [Product Detail — Sales Availability by Channel](./product-detail).
:::
