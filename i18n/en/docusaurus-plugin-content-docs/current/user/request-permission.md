# Request Permission

OMS only shows you data within the **brand × corporation** scope granted to your account. If you can't see the data you need, you must request permission for that scope and obtain an administrator's approval.

---

## How to Request

1. Open the **Request Permission** window.
2. In the **Brand–Corp matrix**, check the **brand × corporation** combinations you want to access.
   - The selectable corporations differ by brand.

   | Brand | Selectable Corporations |
   |--------|------------------|
   | GM | KR, US, JP, AU, SG, TW |
   | TAM | KR, JP |
   | ATS | KR |
   | NUF | KR |

3. Enter the reason you need the permission in **Reason**.
4. Submit the request with **"Submit"**.

---

## After Requesting

- The requested permission enters the **Request (awaiting approval)** status.
- Once an administrator approves it, you will be able to see data within that scope.
- It is normal for the data to remain hidden until approval.

:::tip
You can select and request multiple combinations at once. It's convenient to map out your area of responsibility in advance and apply for everything in a single request.
:::
