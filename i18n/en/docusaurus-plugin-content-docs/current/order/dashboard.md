# Dashboard (Overview)

The **Order → Overview** menu on the left is the first screen you see after logging in. It gives you an **at-a-glance, status-by-status** view of the orders, shipments, and claims that currently need to be processed.

## Screen Layout

The dashboard is divided into two tabs.

| Tab | What it shows |
|----|---------------|
| **ORDER** | Order, Shipment, and Store Pickup status |
| **CLAIM** | Return, Exchange, and Reshipment status |

Within each area, the cards are grouped by processing stage.

- **Awaiting**: Waiting to be processed (e.g., Pending)
- **In Progress**: Being processed
- **Finalized**: Completed, canceled, or otherwise closed items (last 30 days)

For example, on the ORDER tab, `Awaiting > Pending: 3` on the Order card means "there are 3 orders awaiting confirmation."

---

## How to Use

1. Select the **ORDER** or **CLAIM** tab at the top.
2. Click the status card you want to view (e.g., `Awaiting`, `In Progress`).
3. A **filtered order list** for that status appears immediately at the bottom of the screen. With a single click, you can pull up just the items you need to handle right now.

### Other Buttons

| Button | Location | Function |
|------|------|------|
| **Refresh** | Top right | Refreshes to the latest totals (also displays the `Updated at` timestamp) |
| **Show more / Show less** | Bottom center | Expands or collapses the detailed status items |

:::tip Your starting point for the day
Checking the **Awaiting / In Progress** counts on the dashboard each morning gives you a quick read on your workload for the day. Click a number to jump straight to the list.
:::

:::note Scope of the figures
The dashboard figures only count data within the **Brand & Corp** scope selected in the header. If a number looks off, check your header selection first.
:::
