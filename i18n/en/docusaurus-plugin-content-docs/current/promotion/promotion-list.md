# Promotion List

From the left menu **Promotion → Promotion List**, you can review and search promotions, and register, copy, or export them. A promotion refers to a sales incentive provided based on order conditions, such as a **GWP (free gift)** or **packaging benefit**.

---

## Searching for Promotions

| Search Item | Description |
|-----------|------|
| **Date Type** | Period basis — Start Date / End Date |
| **Period** | Period (`Today` `Week` `Month` `3 Month` shortcuts provided) |
| **Status** | Upcoming / Active / Expired / Draft |
| **Channel** | Applied channel |
| Search term | ID / Title / Created By / GWP Name / GWP SAP Code / target product name or code |

### List Columns

| Column | Description |
|------|------|
| ID | Promotion number |
| Brand / Corp | Brand / corporation |
| **Title** | Click to go to the detail page |
| **Status** | Status badge (Active=green, Upcoming=orange, Expired=gray, Draft=blue) |
| Trigger Type | Type of trigger condition |
| Trigger Channels | Applied channels |
| Start / End Date | Start and end date/time |
| Created By / Created At | Creator / creation date |
| **Actions** | Copy icon (*Copy Promotion*) |

---

## Meaning of Status

| Status | Meaning |
|------|------|
| **Draft** | Being drafted (not yet applied) |
| **Upcoming** | Approved, waiting for the start date |
| **Active** | Currently in progress |
| **Expired** | Ended |

---

## Other Tasks

| Button | Location | Function |
|------|------|------|
| **Refresh** | Top right | Refresh the list |
| **Export** | Top right | Download the promotion list to Excel (`IIC_OMS_Promotion_List_YYYYMMDD_HHmm.xlsx`) |
| **Copy (icon)** | Right of the row | Copy an existing promotion to create a new one |

For how to register and edit promotions, see [Create / Edit Promotion](./promotion-create).
