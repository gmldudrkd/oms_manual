# 권한 요청 (Request Permission)

OMS는 사용자에게 부여된 **브랜드 × 법인** 범위의 데이터만 보여줍니다. 필요한 데이터가 안 보이면 해당 범위의 권한을 요청하고 관리자의 승인을 받아야 합니다.

---

## 요청 방법

1. **권한 요청(Request Permission)** 창을 엽니다.
2. **Brand–Corp 매트릭스**에서 접근하려는 **브랜드 × 법인** 조합을 체크합니다.
   - 브랜드별로 선택 가능한 법인이 다릅니다.

   | 브랜드 | 선택 가능한 법인 |
   |--------|------------------|
   | GM | KR, US, JP, AU, SG, TW |
   | TAM | KR, JP |
   | ATS | KR |
   | NUF | KR |

3. **Reason(사유)**에 권한이 필요한 이유를 입력합니다.
4. **"Submit"**으로 요청을 제출합니다.

---

## 요청 후

- 요청한 권한은 **Request(승인 대기)** 상태가 됩니다.
- 관리자가 승인하면 해당 범위의 데이터를 볼 수 있게 됩니다.
- 승인 전까지는 데이터가 보이지 않는 것이 정상입니다.

:::tip
필요한 조합을 한 번에 여러 개 선택해 요청할 수 있습니다. 담당 업무 범위를 미리 정리해 한 번에 신청하면 편리합니다.
:::
