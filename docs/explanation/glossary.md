# 용어 사전

| 용어 | 영문 | 설명 |
|------|------|------|
| 주문 | Order | 고객이 상품을 구매한 건 |
| 출고 | Shipment | 상품을 포장하여 배송하는 건 |
| 매장 픽업 | Store Pickup | 고객이 매장에서 직접 수령 |
| 반품 | Return | 배송된 상품을 반환받는 건 |
| 교환 | Exchange | 상품을 반환받고 다른 상품을 보내는 건 |
| 클레임 | Claim | 주문에 대한 사후 요청 (취소/반품/교환/재출고) |
| 재출고 | Reshipment | 출고 실패 후 다시 배송하는 건 |
| 재고 | Stock | 판매 가능한 상품 수량 |
| 채널 재고 | Channel Stock | 특정 판매 채널에 분배된 재고 |
| 가용재고 | Available Stock | 현재 판매 가능한 수량 (= 분배 - 사용 - 출고) |
| 미분배 재고 | Undistributed | 어떤 채널에도 분배되지 않은 잔여 재고 |
| 안전재고 | Safety Stock | 예비로 확보해두는 최소 재고 수량 |
| 선주문 | Preorder | 재고 입고 전 사전 주문 |
| SKU | SKU | 상품의 고유 식별 코드 (Stock Keeping Unit) |
| UPC | UPC | 바코드 (Universal Product Code) |
| SAP Code | SAP Code | SAP ERP 시스템의 상품 코드 (= productCode) |
| 원주문번호 | Origin Order No | 판매 채널에서 부여한 원본 주문번호 |
| 구매번호 | Purchase No | 결제 관련 고유 번호 |
| 피킹 | Picking | 물류 센터에서 상품을 선반에서 꺼내는 작업 |
| 검수 | Inspection | 반품/교환 상품의 상태를 확인하고 등급을 부여하는 과정 |
| 마스킹 | Masking | 개인정보를 일부 가려서 표시하는 것 |
| WMS | WMS | 물류 관리 시스템 (Warehouse Management System) |
| ERP | ERP | 전사적 자원관리 시스템 (SAP) |
| PIM | PIM | 상품 정보 관리 시스템 (Product Information Management) |
| BO | BO | 관리자 웹 애플리케이션 (Back Office) |
| 법인 | Corporation | 국가별 운영 법인 (KR, US, CA 등) |
| 분배율 | Distribution Rate | 채널별 재고 분배 비율 (%) |
| 번들 상품 | Bundle Product | 여러 단일 상품을 하나의 세트로 묶은 상품 |
| 구성품 | Component | 번들 상품을 구성하는 개별 상품 |
