# 정책

## 개인정보 보호 정책

OMS는 고객의 개인정보를 다음과 같이 보호합니다:

- **마스킹 처리**: 이름, 이메일, 전화번호는 화면에서 마스킹(일부 문자를 ***로 대체)되어 표시
- **접근 로그**: 주문 상세 조회 및 엑셀 다운로드 시 개인정보 조회 로그가 자동 기록됨
- **채널 기반 접근 제어**: 사용자는 권한이 부여된 채널의 데이터만 접근 가능
- **엑셀 암호화**: 다운로드되는 엑셀 파일은 암호로 보호됨

## 재고 관리 정책

- **과매도(Overselling) 방지**: 채널 가용재고가 0 이하이면 자동으로 판매 중단
- **동시성 제어**: 동일 상품에 대한 동시 주문 시 비관적 락(Pessimistic Lock)으로 재고 정합성 보장
- **자동 동기화**: ERP(SAP)와 정기적으로 재고 동기화
- **안전 재고**: 최소 보유 수량 설정으로 예기치 않은 품절 방지
