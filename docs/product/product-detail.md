# 상품 상세 (Product Detail)

상품 목록에서 **SKU Code**를 클릭하면 상품 상세 화면이 열립니다. 여기서 가격·이미지·상품정보·채널 판매 가능 여부를 편집합니다.

---

## 화면 구성

상세 화면은 위에서부터 아래 순서로 구성됩니다.

| 섹션 | 내용 | 편집 |
|------|------|------|
| **Product Master** | SKU·SAP 코드, 상품명, 모델코드 등 기본정보 | 조회 전용 |
| **Price Master** | 통화별 판매 단가 | ✅ (모든 단가 입력 필수) |
| **Paid Packaging Options** | 유료 패키징 옵션별 가격 | ✅ (모든 가격 입력 필수) |
| **Images** | 상품 이미지 업로드·순서 관리 | ✅ (최소 1장, 크기 기준 충족) |
| **Product Information** | 상품 상세 정보 폼 | ✅ (필수 항목 입력) |
| **Sales Availability by Channel** | 채널별 판매 노출 ON/OFF | ✅ |

---

## 편집과 저장

1. 각 섹션의 값을 입력·수정합니다.
2. **이미지**는 파일 선택 또는 드래그 앤 드롭으로 추가하고, 표시 순서를 조정합니다.
3. **"Save"** 버튼으로 저장합니다.
4. 저장 시 다음을 자동 검증합니다.
   - 모든 가격(Price / Packaging) 입력 여부
   - 상품정보 필수 항목 입력 여부
   - 이미지 최소 1장 및 크기 기준
5. 저장에 성공하면 완료 알림이 표시됩니다.

:::note 이미지 반영 지연
새 이미지를 업로드하면 먼저 저장소에 업로드된 뒤 상품에 반영됩니다. 저장 직후 이미지가 바로 안 보일 수 있으며, 잠시 후 새로고침하면 표시됩니다.
:::

:::tip 상품정보 완료 상태
필수 정보가 모두 채워지면 [상품 목록](./product-list)의 **Product Info Status**가 **Complete**로 바뀝니다. Incomplete 상품은 이 화면에서 누락 항목을 채우세요.
:::
