# 채널 상품 목록 (Channel Product List)

좌측 메뉴 **Product → Channel Product List**에서 **채널별로 상품이 어떻게 노출되는지**를 확인합니다. 상품 목록(Product List)이 상품 마스터 관점이라면, 이 화면은 **채널 판매 관점**입니다.

---

## 조회

| 검색 항목 | 입력 방법 |
|-----------|-----------|
| **SKU / SAP / Model / UPC Code** | 줄바꿈으로 여러 건 입력 |
| **SAP Name** | 상품명 |
| **Channel** | 채널 복수 선택 |
| **Channel Availability** | 채널 판매 가능 여부 — Yes / No |
| **Product Info Status** | Complete / Incomplete |
| **Product Type** | Single / Bundle |

### 목록 컬럼

| 컬럼 | 설명 |
|------|------|
| Product Type | Single / Bundle |
| **SKU Code** | 클릭하면 [상품 상세](./product-detail)로 이동 |
| SAP Code / Quantity / SAP Name / Model Code / UPC Code | 상품 기본정보 |
| **Channel** | 채널명 |
| **Channel Availability** | 해당 채널에서 판매 가능한지(Yes/No) |
| **Product Info Status** | 상품정보 완료 여부 |

:::tip
특정 채널에서 상품이 판매되지 않는다면, 이 화면에서 **Channel Availability**가 No인지 확인하세요. 채널 노출은 [상품 상세 — Sales Availability by Channel](./product-detail)에서 변경합니다.
:::
