# 화면 둘러보기

OMS 화면은 **상단 헤더 + 좌측 메뉴 + 본문**의 세 영역으로 구성됩니다. 모든 화면이 같은 패턴을 따르므로, 한 번 익히면 어떤 메뉴에서도 똑같이 사용할 수 있습니다.

---

## 상단 헤더

화면 맨 위 갈색 바에는 운영자가 자주 쓰는 전역 설정이 모여 있습니다.

| 항목 | 위치 | 기능 |
|------|------|------|
| **Brand & Corp** | 가운데 | 작업 대상 **브랜드 × 법인**을 선택. 선택한 조합의 데이터만 화면에 표시됨 |
| **Timezone** | Brand & Corp 옆 | 날짜·시간을 표시할 시간대 선택 (예: Asia/Seoul) |
| **계정 메뉴** | 오른쪽 끝 | 내 ID 표시, **"Log out"** |

### Brand & Corp 전환

가장 자주 쓰는 기능입니다. 헤더의 **Brand & Corp** 영역을 클릭하면 내가 권한을 가진 브랜드·법인 목록이 나타납니다. 원하는 조합을 선택하면 주문·재고·상품 화면이 그 범위로 즉시 갱신됩니다.

- 주문 목록·채널 목록 화면 등에서는 **여러 조합을 동시에 선택(복수 선택)**할 수 있습니다. 모두 선택하면 **"ALL"**로 표시됩니다.
- 일부 상세 화면에서는 단일 선택만 가능합니다.

:::

### Timezone(시간대)

화면에 표시되는 모든 날짜·시간은 여기서 선택한 시간대 기준으로 보입니다. 해외 법인 주문을 다룰 때 현지 시간으로 보려면 시간대를 바꾸세요. 선택값은 브라우저에 저장되어 다음 접속 때도 유지됩니다.

## 좌측 메뉴

좌측 메뉴는 업무 영역별로 그룹화되어 있습니다.

| 메뉴 그룹 | 하위 메뉴 | 매뉴얼 |
|-----------|-----------|--------|
| **Order** | Overview, Order List, Return List, Exchange List, Reshipment List, Export | [주문](../order/dashboard) |
| **Stock** | Overview, Distribution Setting, History | [재고](../stock/overview) |
| **Product** | Product List, Channel Product List | [상품](../product/product-list) |
| **Channel** | Channel List | [채널](../channel/channel-list) |
| **Promotion** | Promotion List | [프로모션](../promotion/promotion-list) |
| **User** | User List | [사용자](../user/user-list) |

각 그룹 제목을 클릭하면 하위 메뉴가 접히거나 펼쳐집니다.

## 공통 UI 패턴

모든 목록 화면은 동일한 구성 요소를 사용합니다. 아래 패턴만 익히면 됩니다.

### 검색 폼 (Search Form)

화면 상단의 검색 영역입니다.

- **날짜 범위(Period)**: 달력으로 기간 선택. `Today`, `Week`, `Month`, `3 Month`, `Reset` 같은 단축 버튼 제공
- **상태 필터**: 여러 상태를 체크박스로 복수 선택
- **검색어**: 드롭다운으로 검색 기준(주문번호·이름·이메일 등)을 고른 뒤 키워드 입력. 여러 건을 줄바꿈으로 한 번에 검색 가능(보통 최대 100건)
- **"Search"**: 검색 실행 / **"Reset"**: 조건 초기화

### 데이터 그리드 (목록 표)

검색 결과가 표 형태로 나타납니다.

- 각 행 왼쪽의 **체크박스**로 여러 건을 선택해 **일괄 작업(Bulk)**을 수행
- 파란색 **링크(주문번호 등)**를 클릭하면 상세 화면으로 이동
- 상태는 **색상 칩(chip)**으로 구분 표시
- 우측 상단에 **"Refresh"** 버튼으로 최신 데이터 갱신

### 모달(팝업) — 작업 처리

취소·환불·수정 등 중요한 작업은 **모달 팝업**에서 처리합니다. 공통 흐름은 다음과 같습니다.

> **행 선택 → 작업 버튼 클릭 → 모달에서 사유·수량 등 입력 → 확인 버튼**

모달에서 확인을 누르면 처리 결과가 화면 하단의 **알림(스낵바)**으로 표시되고, 목록이 자동으로 갱신됩니다.

:::warning
취소·환불·삭제처럼 되돌리기 어려운 작업은 확인 버튼을 누르기 전에 **선택한 항목과 사유**를 반드시 다시 확인하세요.
:::
