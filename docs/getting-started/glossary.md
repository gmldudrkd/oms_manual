# 용어집

매뉴얼과 화면에 자주 나오는 용어를 정리했습니다. 화면 표기는 영어이므로 영어 용어를 기준으로 정렬했습니다.

## 조직 · 구조

| 용어 | 설명 |
|------|------|
| **Brand** | 판매 브랜드 (GENTLE MONSTER, TAMBURINS, ATIISSU, NUFLAAT 등) |
| **Corp** | 판매·정산 국가 법인 (KR, US, JP, CA, TW, SG, AU 등) |
| **Channel** | 실제 판매 창구. 공식몰(Official) 또는 마켓플레이스(KAKAO, SSG, FARFETCH 등) |
| **Role** | 역할. Admin / Manager / Request / None |

## 상품 · 재고

| 용어 | 설명 |
|------|------|
| **SKU** | 상품의 최소 판매 단위(색상·사이즈 구분 포함) 코드 |
| **SAP Code** | SAP(ERP)에서 관리하는 상품 코드 |
| **UPC Code** | 바코드(상품 식별) 코드 |
| **Bundle** | 여러 단품(Single)을 묶어 만든 세트 상품 |
| **Online Stock** | 브랜드·법인 단위의 전체 가용 재고(채널에 분배되기 전) |
| **Channel Stock** | 특정 채널에 분배된 재고 |
| **Safety Stock(안전재고)** | 품절을 막기 위해 따로 빼두는 최소 보유 수량 |
| **Distribution(분배)** | 온라인 재고를 채널별로 나눠 배정하는 것 |
| **Pre-order(프리오더)** | 입고 전 미리 판매하는 사전주문 수량 |

## 주문 · 출고

| 용어 | 설명 |
|------|------|
| **Order** | 고객 주문 건 |
| **Order No** | OMS 주문 번호 |
| **Origin/Purchase No** | 판매 채널의 원주문 번호 |
| **Shipment** | 출고 건(포장·배송 단위). 한 주문이 여러 출고로 나뉠 수 있음 |
| **Store Pickup** | 매장 수령 주문 |
| **Receive Method** | 수령 방법. Delivery(배송) 또는 Store Pickup(매장픽업) |
| **Order Type** | 주문 유형. NORMAL / GIFT / RX |
| **Tracking No** | 택배 송장 번호 |
| **WMS** | 물류창고 관리 시스템(피킹·포장·배송 수행) |

## 클레임(반품/교환/취소)

| 용어 | 설명 |
|------|------|
| **Claim** | 취소·반품·교환·재출고 요청을 통칭 |
| **Return** | 반품(상품 회수 + 환불) |
| **Exchange** | 교환(기존 상품 회수 + 새 상품 발송) |
| **Reshipment** | 재출고(출고 실패·분실 건 재발송) |
| **Pickup** | 반품·교환을 위한 상품 회수 |
| **Inspection / Grading** | 회수 상품 검수 및 등급 판정(A/B/C) |
| **Force Refund** | 검수 없이 즉시 환불 처리 |
| **Fault** | 귀책. CUSTOMER(고객 귀책) / OPERATION(운영 귀책) |

## 시스템

| 용어 | 설명 |
|------|------|
| **SAP (ERP)** | 정산·재무 시스템 |
| **PIM** | 상품 정보 원천 시스템 |
| **Adapter** | OMS와 외부 채널·창고·ERP를 연결하는 중계 시스템 |
| **Export** | 주문 등 데이터를 암호화된 엑셀로 내려받는 기능 |
