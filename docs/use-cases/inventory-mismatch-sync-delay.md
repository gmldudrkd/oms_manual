# 재고 불일치 / 연동 지연 시나리오

## 상황
ERP(SAP) 재고와 OMS 채널 재고가 맞지 않는 경우.

## 원인
- ERP에서 재고 동기화가 지연될 수 있음
- 동시 주문으로 인한 일시적 수량 차이
- 물류 센터 실물 재고와 시스템 재고 차이

## 확인 방법
1. **재고 변동 이력** 조회 → 최근 변경 확인
2. **미분배 재고** 조회 → 아직 채널에 분배되지 않은 수량 확인
3. **채널별 가용재고** 조회 → 실제 판매 가능 수량 확인

## 대응 방법
- 재고 동기화 대기 (일반적으로 자동 복구)
- 채널 간 재고 이전(Transfer)으로 불균형 해소
- 선주문(Preorder) 설정으로 임시 판매 허용
- 미분배 재고를 해당 채널에 수동 배정
