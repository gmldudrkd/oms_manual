# 부분 검수 환불 (Partial Inspection Refund)

> **상황**: 반품으로 여러 개를 회수했는데, 일부는 재판매 가능(A)하고 일부는 하자(B/C)가 있어 등급이 섞여 있습니다.

## 대응 순서

1. 반품이 **Received**(입고/검수) 상태가 되면 RETURN 탭의 **Product Inspection Result**를 엽니다.
2. **회수된 수량마다 등급을 개별 지정**합니다.

   | 등급 | 의미 | 재고 처리 |
   |------|------|-----------|
   | **A** | 재판매 가능 | 정상 재고 복귀 |
   | **B** | 경미한 하자 | 별도 재고 풀 |
   | **C** | 판매 불가 | 폐기 |

3. **모든 수량에 등급을 지정**한 뒤 **"Refund"**로 환불을 확정합니다.

## 체크 포인트

- 등급이 하나라도 비어 있으면 환불을 진행할 수 없습니다.
- 등급과 **귀책(Fault)**에 따라 환불 금액이 달라질 수 있습니다(브랜드 정책).
- 검수 없이 즉시 환불해야 하는 건은 **Force Refund**로 처리하며, 반품 카드에 **"FORCE REFUND"** 배지가 표시됩니다.

자세한 검수 절차는 [반품 처리 — 검수 및 환불](../order/return#3-입고-확인-후-검수-및-환불-refund)을 참고하세요.
