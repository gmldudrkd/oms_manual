# 채널 관리

## 채널이란?

채널은 상품을 판매하는 **판매 경로**입니다. 각 채널은 특정 브랜드와 법인에 소속되며, 독립적으로 재고와 주문을 관리합니다.

## 채널 목록 조회

등록된 모든 판매 채널을 조회할 수 있습니다.

**채널 정보:**

| 항목 | 설명 |
|------|------|
| 채널명 | 판매 채널 이름 |
| 채널 유형 코드 | 채널 고유 코드 (예: E6001, US1004) |
| SAP 코드 | ERP 시스템의 채널 코드 |
| 브랜드 | 소속 브랜드 (ATiissu, Nuflaat, GM) |
| 법인 | 소속 법인 (KR, US, CA 등) |
| 카테고리 | 채널 분류 |
| 활성 상태 | 활성 / 비활성 |

## 활성 채널 조회

브랜드와 법인으로 필터링하여 현재 활성화된 채널만 조회할 수 있습니다.

## 채널 등록

> **권한**: ADMIN 이상 역할 필요

새 판매 채널을 등록하는 절차:

1. **채널 관리** 메뉴 이동
2. **채널 등록** 클릭
3. 채널 정보 입력:
   - 채널명
   - 채널 유형 코드
   - 브랜드 선택
   - 법인 선택
   - 카테고리 설정
4. 등록 완료

> **주의**: 동일한 채널 유형 코드가 이미 등록되어 있으면 `ChannelType already registered` 오류가 발생합니다.

## 채널 수정

기존 채널의 정보를 수정할 수 있습니다:

- 채널명 변경
- 카테고리 변경
- 활성/비활성 상태 변경

> **주의**: 채널 비활성화 시 해당 채널에 잔여 재고가 있으면 `Certain inventory remains on the channel` 오류가 발생합니다. 재고를 다른 채널로 이전한 후 비활성화해야 합니다.
