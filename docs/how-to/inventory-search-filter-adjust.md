# 재고 조회 / 필터 / 조정 요청

## 재고 구조

OMS의 재고는 두 가지 레벨로 관리됩니다:

| 구분 | 설명 |
|------|------|
| **Online Stock** | ERP(SAP)에서 동기화되는 마스터 재고 |
| **Channel Stock** | 각 판매 채널에 분배된 재고 |

```
ERP 전체 재고 (Online Stock)
    ├── ATiissu Official: 100개 (Channel Stock)
    ├── Nuflaat Official: 50개 (Channel Stock)
    └── 미분배: 30개 (Undistributed)
```

## 재고 수량 개념

| 수량 | 의미 | 계산 |
|------|------|------|
| 분배수량(Distributed) | 해당 채널에 할당된 총 수량 | ERP에서 설정 |
| 사용수량(Used) | 주문으로 예약된 수량 | 주문 생성 시 증가 |
| 출고수량(Shipped) | 실제 배송된 수량 | 출고 시 증가 |
| **가용수량(Available)** | **현재 판매 가능 수량** | = 분배 - 사용 - 출고 |
| 선주문수량(Preorder) | 사전 주문 허용 수량 | 별도 설정 |

> **채널 가용재고 조회 API (OMS-2002)**: 특정 채널의 상품별 가용 재고를 한 번에 조회할 수 있는 API가 추가되었습니다.

## 재고 조회 화면

| 화면 | 설명 |
|------|------|
| 온라인 재고 대시보드 | 전체 SKU별 온라인 재고 현황 |
| 채널 재고 대시보드 | 채널별 SKU 재고 현황 |
| 재고 변동 이력 | SKU별 재고 변경 기록 (주문 생성, 취소, 출고 등) |
| 미분배 재고 조회 | 아직 채널에 분배되지 않은 수량 |

## 재고 이동 (Transfer)

채널 간 재고를 이전할 수 있습니다:
1. 출발 채널, 도착 채널 선택
2. 이전할 수량 입력
3. 이전 완료 → 양쪽 채널의 가용수량 자동 변경

> 동일 브랜드/법인 내의 채널 간에서만 이전 가능합니다.

## 선주문(Preorder) 설정

재고가 없는 상품도 사전 주문을 받을 수 있습니다:
1. 채널별 선주문 수량 설정
2. 선주문 만료일 설정
3. 만료 전까지 가용수량에 선주문 수량이 포함됨
