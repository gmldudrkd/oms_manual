# 주문 조회 / 필터 / 다운로드

## 주문 목록 조회

**사용 가능한 검색 필터:**

| 필터 | 설명 | 입력 제한 |
|------|------|----------|
| 원주문 번호 | 판매 채널의 주문번호 | 최대 100건 |
| 출고 번호 | 배송 건의 번호 | 최대 100건 |
| SKU | 상품 코드 | 최대 100건 |
| 주문자명 | 주문한 고객 이름 | 부분 검색 가능 |
| 주문자 이메일 | 고객 이메일 | 부분 검색 가능 |
| 주문자 전화번호 | 고객 전화번호 | 부분 검색 가능 |
| 주문 유형 | NORMAL / GIFT / RX | 복수 선택 가능 |
| 주문 상태 | PENDING ~ COMPLETED | 복수 선택 가능 |
| 출고 상태 | PICKING_REQUESTED ~ DELIVERED | 복수 선택 가능 |
| 매장픽업 상태 | 매장 픽업 진행 상태 | 복수 선택 가능 |
| 채널 | 판매 채널 | 권한 범위 내에서만 |
| 수령 방법 | 배송 / 매장픽업 | 복수 선택 가능 |
| 태그 | PREORDER 등 | 복수 선택 가능 |
| 날짜 범위 | 주문 접수일 기준 | 완료 건 조회 시 필수 |

> **주의**: 완료/취소/삭제된 주문을 조회할 때는 반드시 날짜 범위를 설정해야 합니다.

> **매장픽업 상태 필터 (OMS-2022)**: 매장 픽업 주문의 진행 상태별로 필터링할 수 있습니다.

> **수령 방법 필터 (OMS-2017)**: 배송 주문과 매장 픽업 주문을 구분하여 조회할 수 있습니다.

> **태그 필터 (OMS-2045)**: `PREORDER`(사전주문) 태그가 부여된 주문만 조회할 수 있습니다.

## 주문 상세 조회

주문을 클릭하면 다음 정보를 확인할 수 있습니다:

- **헤더**: 주문번호, 브랜드, 채널, 주문유형, 수령방법, 태그, 주문일시
- **주문자/수령인 정보**: 이름, 연락처, 주소 (마스킹 처리)
- **결제 정보**: 결제수단, 결제금액, 세금, 관세, 배송비, 통화
- **주문 상품**: SKU, 상품코드, 상품명, 카테고리, 수량별 상태
  - 주문수량, 할당수량, 취소수량, 출고수량, 배송완료수량, 반품수량, 재출고수량
- **관련 출고 목록**: 해당 주문에 연결된 출고 건들
- **관련 매장픽업**: 매장 픽업 정보 (해당 시)
- **환불 이력**: 반품/교환 관련 환불 결제 내역

> **응답 필드 확장 (OMS-2000, OMS-2001)**: 재출고 수량, 매장 픽업 정보, 환불 결제 내역 등이 추가되었습니다.

## 주문 이력 조회

주문의 전체 변경 이력을 시간순으로 확인할 수 있습니다:

- 주문 상태 변경 이력
- 출고 이벤트 이력
- 매장 픽업 이벤트 이력
- 반품/교환 이벤트 이력
- 재출고 이벤트 이력

> **이력 확장 (OMS-1996)**: 매장 픽업 및 재출고 이력 정보가 주문 이력에 추가되었습니다.

## 주문 다운로드 (Export)

주문 데이터를 엑셀 파일로 다운로드할 수 있습니다.

**다운로드 유형:**

| 유형 | 설명 |
|------|------|
| ALL | 전체 주문 데이터 |
| ORDER | 주문 정보만 |
| SHIPMENT | 출고 정보 |
| STORE_PICKUP | 매장 픽업 정보 |
| ORDER_CANCEL | 취소된 주문 |
| RETURN | 반품 데이터 |
| EXCHANGE | 교환 데이터 |
| RESHIPMENT | 재출고 데이터 |

> **다운로드 유형 추가 (OMS-1999)**: STORE_PICKUP, RESHIPMENT 유형이 새로 추가되었습니다.

> **개인정보 보호**: 엑셀 파일은 암호화되어 다운로드되며, 다운로드 시 개인정보 조회 로그가 자동 기록됩니다.
