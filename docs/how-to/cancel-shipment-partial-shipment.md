# 출고 취소 / 부분출고

## 출고 취소

출고가 아직 배송사에 전달되지 않은 경우 취소할 수 있습니다.

**취소 가능 조건:**
- 출고 상태가 `피킹 거절(PICKING_REJECTED)` 상태일 때
- 취소 사유 입력 필수

**절차:**
1. 주문 상세에서 해당 출고 건 선택
2. **출고 취소** 버튼 클릭
3. 취소 사유 입력
4. 취소 완료 → 할당된 재고가 자동 복원됨

> **배송 중(SHIPPED) 상태에서는 취소가 불가능합니다.** 이 경우 반품 처리를 진행해야 합니다.

## 부분 출고

주문 내 일부 상품만 먼저 출고할 수 있습니다.

**절차:**
1. 주문 상세에서 **부분 출고 요청** 클릭
2. 출고할 상품과 수량을 선택
3. 부분 출고 요청 완료

**상태 변화:**
- 부분 출고 요청 → 주문 상태가 `부분 출고 요청(PARTIAL_SHIPMENT_REQUESTED)`으로 변경
- 나머지 상품 출고 요청 시 → `출고 요청(SHIPMENT_REQUESTED)`으로 변경

> **분할 배송**: 하나의 주문이 여러 출고 건으로 나뉠 수 있습니다. 모든 출고 건이 배송 완료되어야 주문이 완료됩니다.
