# 클레임 등록

## 클레임이란?

클레임은 **주문에 대한 사후 요청**을 관리하는 단위입니다. 반품, 교환, 취소, 재출고 요청이 모두 클레임으로 관리됩니다.

## 클레임 유형

| 유형 | 설명 |
|------|------|
| 취소(CANCEL) | 주문 또는 일부 상품 취소 |
| 반품(RETURN) | 배송된 상품 반환 → 환불 |
| 반품 강제환불(RETURN_FORCE_REFUND) | 특수 반품 → 즉시 환불 |
| 교환(EXCHANGE) | 배송된 상품 반환 → 다른 상품 출고 |
| 재출고(RESHIPMENT) | 출고 실패 건 재배송 |

## 책임 구분

| 구분 | 설명 | 예시 |
|------|------|------|
| 고객 책임(CUSTOMER) | 고객의 사유로 발생 | 단순 변심, 사이즈 불일치 |
| 운영 책임(OPERATION) | 회사/배송사의 사유 | 오배송, 상품 불량, 배송 파손 |

## 클레임 등록 절차

1. 주문 상세 화면 이동
2. **클레임 등록** 클릭
3. 유형, 사유, 책임 구분 선택
4. 대상 상품 및 수량 선택
5. 등록 완료

> 클레임이 등록되면 유형에 따라 **반품(Return)**, **교환(Exchange)**, 또는 **재출고(Reshipment)** 엔티티가 자동 생성됩니다.
