# 화면 / 메뉴별 필드 정의

## 주문 목록 화면

| 필드 | 설명 |
|------|------|
| 주문번호 | OMS 내부 주문 ID |
| 원주문번호 | 판매 채널에서 부여한 주문번호 |
| 구매번호 | 결제 관련 고유 번호 |
| 브랜드 | GM / TAM / NUD / NUF / ATS |
| 채널 | 판매 채널명 |
| 주문유형 | NORMAL / GIFT / RX |
| 수령방법 | ADDRESS_DELIVERY / STORE_PICKUP |
| 태그 | PREORDER 등 |
| 주문상태 | PENDING ~ COMPLETED |
| 주문일시 | 주문 접수 시각 |
| 주문자명 | 고객 이름 (마스킹) |

## 주문 상세 화면 - 상품 정보

| 필드 | 설명 |
|------|------|
| SKU | 상품 고유 코드 |
| 상품코드 | 내부 상품 코드 (SAP Code) |
| 상품명 | 상품 이름 |
| UPC | 바코드 |
| 카테고리 | 상품 분류 |
| 주문수량 | 고객이 주문한 수량 |
| 할당수량 | 재고에서 확보한 수량 |
| 취소수량 | 취소된 수량 |
| 출고수량 | 출고 처리된 수량 |
| 배송완료수량 | 고객에게 전달된 수량 |
| 반품수량 | 반품 처리된 수량 |
| 재출고수량 | 재출고 처리된 수량 |
| 단가 | 상품 1개 가격 |
| 소계 | 주문수량 x 단가 |

## 반품/교환 상세 화면

| 필드 | 설명 |
|------|------|
| 반품/교환 번호 | 반품 또는 교환 건의 고유 번호 |
| 원주문번호 | 원래 주문의 번호 |
| 상태 | 현재 진행 상태 |
| 클레임 사유 | 반품/교환 사유 |
| 책임 구분 | 고객 책임 / 운영 책임 |
| 수거 배송정보 | 배송사, 송장번호, 추적 URL |
| 검수 결과 | 상품별 검수 등급 (A/B/C) |
| 환불 정보 | 환불 금액 및 상세 (반품의 경우) |
