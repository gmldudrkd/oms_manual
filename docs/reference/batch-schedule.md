# 배치 스케줄표 (Batch Schedule)

OMS에서 정기적으로 실행되는 배치 작업(주문 수집·수신, 클레임 등)의 실행 시간을 정리한 표입니다. 시간은 KST 기준입니다.

:::note
자사몰의 경우 주문, 클레임 모두 실시간으로 연동이 진행됩니다.
:::

## GM KR
| 작업 | 실행 시간 (KST) | 주기 | 설명 |
| --- | --- | --- | --- |
| 카카오 결제완료 주문 수집 | 20분 | 매시 | 결제완료된 카카오 채널 주문 건을 수집합니다. (배송지가 이미 입력된 주문은 샵링커 결제완료 수집 시 접수됩니다.) |
| 카카오 배송지 입력전 취소 확인 | 00,20,40 분 | 매시 | 카카오 채널의 주문확인 전 취소 건을 조회합니다. |
| 샵링커 결제완료 수집 (SIV, SSG, SSF) | 15분 | 매시 | 결제완료 된 SIV, SSG,SSF 채널 주문 건을 수집합니다. |
| 샵링커 주문확정 처리 (SIV, SSG, SSF) | 00,20,40 분 | 매시 | 주문 후 1시간이 지난 건에 대해 주문확인(배송준비중) 처리를 합니다. |
| 샵링커 클레임 수집 (SIV, SSG, SSF) | 25분 | 매시 | SIV, SSG,SSF 채널에 접수된 취소,반품 클레임을 수집합니다. |
| 파페치 결제완료 주문 수집 | 30분 | 매시 | 결제완료된 파페치 채널 주문 건을 수집합니다. |
| 파페치 주문확정/송장생성 처리 | 00,20,40 분 | 매시 | 주문 후 1시간이 지난 파페치 주문 건에 대해 주문확인 처리를 하고 송장을 업데이트 합니다. |
| 파페치 출고요청 처리 | 40분 | 매시 | 주문 확정 처리된 파페치 주문 건의 출고요청 처리를 합니다. |
| 파페치 클레임 수집 | 35분 | 매시 | 파페치 채널에 접수된 취소,반품 클레임을 수집합니다. |
