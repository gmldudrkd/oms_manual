# 자주 보는 안내·에러 메시지

운영 중 자주 마주치는 화면 안내·확인 메시지와 대응 방법입니다.

## 권한 / 데이터 표시

| 메시지/현상 | 의미 | 대응 |
|-------------|------|------|
| 목록이 비어 있음 | 해당 Brand & Corp 권한이 없거나, 다른 법인을 보고 있음 | 헤더의 Brand & Corp 확인, 필요 시 [권한 요청](../user/request-permission) |
| 접근 거부(403) | 권한 밖의 데이터에 접근 | 담당 범위 확인, 권한 요청 |

## 주문 / 클레임

| 상황 | 대응 |
|------|------|
| 취소 버튼이 안 보임 | 현재 상태에서 취소 불가 — [상태 코드표](./status-codes) 확인 |
| "Bulk Cancel" 비활성화 | 취소 가능 상태가 아닌 행이 선택됨 |
| 환불(Refund) 진행 불가 | 검수 등급이 일부 비어 있음 — 모든 수량에 등급 지정 |

## 채널 등록

| 메시지 | 의미 | 대응 |
|--------|------|------|
| *"SAP Channel Code (해당 코드) is already registered in OMS."* | 이미 등록된 SAP 채널 코드 | 다른 코드 선택 |
| *"You have unsaved changes..."* | 저장하지 않은 변경분 존재 | 저장 또는 변경 취소 후 이동 |

## Export / 다운로드

| 상황 | 대응 |
|------|------|
| "Export Data" 비활성화 | 기간(Period)이 올바르지 않음 — 시작/종료일 확인 |
| "Excel download" 비활성화 | 비밀번호(4~10자) 또는 목적(Purpose) 미입력 |
| 엑셀 파일이 안 열림 | 다운로드 시 설정한 비밀번호 입력 필요 |

## 프로모션

| 상황 | 대응 |
|------|------|
| 항목 수정 불가 | Active/Upcoming/Expired 상태에서는 일부 항목만 수정 가능 |
| "Delete" 비활성화 | 확인란에 `delete`를 정확히 입력해야 활성화 |

:::tip
대응 방법을 찾기 어려운 문제(연동 지연, 재고 불일치 등)는 [자주 겪는 상황](../use-cases/inventory-mismatch-sync-delay) 챕터를 먼저 확인하고, 해결되지 않으면 시스템 담당자에게 문의하세요.
:::
