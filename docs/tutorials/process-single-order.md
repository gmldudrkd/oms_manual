# 주문 1건 처리

주문 수신부터 배송 완료까지의 전체 과정을 단계별로 안내합니다.

## Step 1. 주문 수신 확인

주문은 판매 채널(자사몰)에서 고객이 결제를 완료하면 OMS에 자동으로 들어옵니다.

1. **주문 목록 조회** 화면으로 이동
2. 검색 필터에서 상태를 `대기(PENDING)`로 설정
3. 방금 들어온 주문을 확인

> **주문이 보이지 않나요?**
> - 내 채널 권한을 확인하세요
> - 검색 기간(날짜)을 확인하세요
> - 필터 조건을 초기화해 보세요

## Step 2. 주문 상세 확인

주문을 클릭하면 아래 정보를 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| 원주문 번호 | 판매 채널에서 부여한 주문번호 |
| 주문 유형 | 일반(NORMAL) / 선물(GIFT) / RX(처방) |
| 수령 방법 | 배송(ADDRESS_DELIVERY) / 매장픽업(STORE_PICKUP) |
| 태그 | PREORDER(사전주문) 등 |
| 주문자 정보 | 이름, 이메일, 전화번호 (개인정보 마스킹 처리됨) |
| 수령인 정보 | 배송 받을 사람의 정보 |
| 결제 정보 | 결제 수단, 금액, 세금, 관세, 배송비 |
| 주문 상품 | SKU, 상품명, 수량별 상태(주문/할당/취소/출고/배송완료/반품/재출고 수량), 가격 |

> **개인정보 보호**: 고객의 이름, 이메일, 전화번호는 마스킹 처리되어 표시됩니다. 상세 정보 조회 시 개인정보 조회 로그가 자동 기록됩니다.

## Step 3. 주문 확정 (PENDING → COLLECTED)

주문이 확정되면 재고가 자동으로 할당됩니다.

- **완전 할당**: 모든 상품의 재고가 충분 → `출고 요청(SHIPMENT_REQUESTED)` 상태로 전환
- **부분 할당**: 일부 상품만 재고 충분 → `부분 확정(PARTLY_CONFIRMED)` 상태로 전환

## Step 4. 출고 처리

재고가 할당되면 물류 센터(WMS)에 출고 지시가 전달됩니다.

```
출고 요청 → 피킹 → 포장 → 배송사 전달 → 배송 중 → 배송 완료
```

| 출고 상태 | 의미 |
|----------|------|
| 피킹 요청(PICKING_REQUESTED) | WMS에 상품 픽업 요청 |
| 피킹 완료(PICKED) | 상품이 선반에서 꺼내짐 |
| 포장 완료(PACKED) | 상품이 포장됨 |
| 배송 중(SHIPPED) | 배송업체에 인도됨, 송장번호 생성 |
| 배송 완료(DELIVERED) | 고객에게 전달됨 |

## Step 5. 배송 완료 → 주문 완료

모든 출고 건이 `배송 완료(DELIVERED)`가 되면, 주문은 자동으로 `완료(COMPLETED)` 상태가 됩니다.
