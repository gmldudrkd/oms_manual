### 7️⃣ Exchange Info 기능 추가

1. 입고 전 교환 출고 기능 추가 → `제품입고 전 선출고 가능`
    

2. 교환 출고 상태 조회 가능

    
3. 교환 클레임 ‘Inspected’ 상태 값 추가
    
    ❓ I**nspected**
    
    - 교환은 입고 후 종결이 아닌 출고까지 완료된 후 종결상태이기에 ‘그레이딩 완료’ 를 ‘검수된’ 상태로 남기도록 변경
    - 교환 출고가 완료되면 그때 Exchanged 상태 변경
    
    🆙 **변경 점**
    
    - AS-IS : 교환 제품 입고 후 Grading 완료 시 ‘Exchanged’
    - TO-BE : 교환 제품 입고 후 Grading 완료 시 ‘Inspected’
        - *채널재고와 교환출고 여부에 따른 Inspected 이후 교환 클레임 상태 변화*
            
            
            | 채널재고 여부 \ 교환 출고 진행 여부 | O | X |
            | --- | --- | --- |
            | O | 교환 출고 출고완료 → Exchanged, 교환 출고 진행 중 → Shipment Requested | Shipment Requested |
            | X | 교환 출고 출고완료 → Exchanged, 교환 출고 진행 중 → Shipment Requested | Inspected |

### 🔟 교환출고, 클레임 출고 등으로 인한 재출고 시 `채널 재고` 사용

🆙 **변경 점**

- AS-IS : 주문에 의한 출고가 아닌 교환, 불량, 분실 등으로 인한 재출고 시 재고 미사용, 무조건 출고 진행
- TO-BE : 주문에 의한 출고가 아닌 교환, 불량, 분실 등으로 인한 재출고 발생 시 채널 재고를 사용

⚠️ **유의점**

- 채널재고가 없을 경우 ‘교환, 불량, 분실 등으로 인한 재출고’ 불가
- 재고를 이동받아 채널재고를 채운 후 수기 출고 진행 필요
